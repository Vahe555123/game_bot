# Архив старых скриптов миграции

Эта папка содержит скрипты миграции, которые были созданы для работы со старой структурой базы данных.

## Причина архивирования

Проект был изначально разработан под одну структуру базы данных (с отдельными полями для UAH, TRL, INR цен и конвертацией в RUB), но затем была использована готовая база данных с другой структурой (товары разделены по регионам: en-ua, en-tr, en-in).

Все модели, API эндпоинты, CRUD операции и шаблоны были обновлены под новую структуру базы данных.

## Список архивированных скриптов

- `fix_currency_rates.py` - исправление курсов валют (старая система)
- `fix_rates_active.py` - активация курсов валют
- `migrate_currency_system.py` - миграция системы валют
- `migrate_database.py` - общая миграция базы данных
- `migrate_psn_fields.py` - миграция PSN полей
- `migrate_to_new_structure.py` - миграция на новую структуру (старая версия)
- `populate_inr_prices.py` - заполнение цен в индийских рупиях
- `test_currency_system.py` - тестирование системы валют
- `update_all_ruble_prices.py` - обновление рублевых цен

## Дата архивирования

14 октября 2025

## Новая структура

Новая база данных использует:
- Разделение товаров по регионам вместо валют
- Простые числовые поля для цен (price, old_price, ps_price, ea_price)
- Информацию о регионе в каждой записи товара (en-ua, en-tr, en-in)
- Без конвертации в рубли на уровне базы данных
